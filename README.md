# currency-converter
# 💱 Currency Converter

This project is a web-based Currency Converter application developed using HTML, CSS, and JavaScript. It enables users to convert an amount from one currency to another using real-time exchange rates. The application provides a simple, fast, and user-friendly interface for performing currency conversions.

The project demonstrates practical implementation of front-end development concepts along with API integration and asynchronous JavaScript.

---

## 🚀 Features

- 💱 Real-time currency conversion
- 🌍 Support for multiple international currencies
- 🔄 Live exchange rates using API integration
- ⚡ Instant calculation and dynamic UI updates
- 📱 Fully responsive design (mobile, tablet, desktop)
- 🎨 Clean and modern user interface

---

## 🛠️ Tech Stack

- **HTML5** – Structure of the application  
- **CSS3** – Styling, layout, and responsiveness  
- **JavaScript (ES6)** – Application logic, API handling, and DOM manipulation

## ⚙️ How It Works

- The user selects the source currency and the target currency.
- The user enters the amount to be converted.
- JavaScript sends a request to a currency exchange API.
- The API returns the latest exchange rate.
- The application calculates and displays the converted amount instantly.

---

## 🎯 Objective

The main objectives of this project are:

- To learn and implement API integration using JavaScript  
- To understand asynchronous programming (fetch, promises, async/await)  
- To practice DOM manipulation and event handling  
- To build a real-world utility application  
- To improve front-end development and UI design skills
